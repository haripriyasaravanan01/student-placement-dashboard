# 🎓 Student Placement Dashboard

## 📌 Project Overview

This Power BI dashboard provides a comprehensive analysis of student placement performance, employability scores, and salary trends. The dashboard helps educational institutions understand placement outcomes and identify factors influencing student employability and salary packages.

The analysis focuses on student demographics, educational background, specialization, work experience, placement status, and salary distribution.

---

## 🎯 Business Problem

Educational institutions need to monitor placement performance and understand which factors contribute to successful student placements.

This dashboard answers key questions such as:

- What is the overall placement rate?
- How many students are placed versus not placed?
- Does work experience improve placement chances?
- Which specialization performs better?
- How does employability score affect salary?
- What is the salary distribution among placed students?

## 📈 Dashboard Features

### 1. Executive Summary
Provides a high-level overview of placement performance including:

- Total Students
- Placed Students
- Placement Rate
- Average Salary
- Highest Salary

### 2. Placement Status Analysis

Visualizes:

- Number of placed students
- Number of non-placed students
- Overall placement percentage

### 3. Employability Score Analysis

Compares average employability scores across specializations:

- Marketing & Finance (Mkt&Fin)
- Marketing & HR (Mkt&HR)

### 4. Work Experience vs Placement Status

Analyzes the relationship between:

- Prior work experience
- Placement outcomes

This helps determine whether experience improves placement opportunities.

### 5. MBA Score vs Salary

Examines how academic performance influences salary packages offered to students.

### 6. Gender Distribution Analysis

Shows student population distribution by gender and supports diversity analysis.

### 7. Interactive Filters

Users can filter the dashboard by:

- Gender
- Degree Type
- Specialization
- Work Experience
