# Student Placement Dashboard

## Overview
This Power BI dashboard provides insights into student placement performance and salary trends.

## Features
- Total Students
- Placement Rate
- Average Salary
- Highest Salary
- Gender Analysis
- Placement Status Analysis
- Employability Score Analysis
- MBA Score vs Salary

## Tools Used
- Power BI
- Excel / CSV
- Data Visualization
